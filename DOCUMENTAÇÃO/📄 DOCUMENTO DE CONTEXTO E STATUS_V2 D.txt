2. VISÃO GERAL DO PROJETO
2.1 O Que Estamos Construindo
Estamos desenvolvendo um sistema web completo para automatizar o processo de assessment de maturidade em Segurança da Informação, baseado na norma ABNT NBR ISO/IEC 27001:2022.

O sistema substitui um processo manual (planilhas Excel, PowerPoint e Word) por uma plataforma digital com:

Landing Page institucional

Sistema de Autenticação com 4 perfis (Admin, Preposto, Consultor, Usuário)

Gestão de Usuários (CRUD completo)

Cadastro e Atribuição dos 93 controles da ISO 27001

Coleta de Respostas dos usuários

Dashboards e Relatórios de maturidade

Multi-tenancy (múltiplas empresas)

Gestão de consultores independentes

4. CONTRATO DE DESENVOLVIMENTO — AS 3 LEIS INVOLÁVEIS
🔒 LEI 1 — MEMORY BANK (DOCUMENTAÇÃO ATIVA)
Nunca confiar apenas na memória de contexto

Antes de alterar QUALQUER linha de código, criar ou atualizar um relatório em docs/audits/

A cada nova interação, reler os .md relevantes antes de agir

🔒 LEI 2 — CORREÇÃO TOTAL (ZERO PARCIALIDADE)
Nunca avançar para o próximo Pilar enquanto TODAS as não-conformidades do Pilar atual não estiverem corrigidas

O relatório .md deve estar marcado como Validado antes de prosseguir

Nunca avançar para o próximo Módulo antes do ciclo completo dos 7 Pilares do módulo atual

🔒 LEI 3 — DIVISÃO DE CARGA (ANTI-TRUNCAMENTO)
Se o plano de execução exceder ~300 linhas de código, dividir em blocos numerados

Ao final de cada bloco, parar e aguardar o comando continuar

Nunca entregar código truncado, com // ... resto igual, // TODO: completar, ou similares

5. METODOLOGIA — OS 7 PILARES
Seguimos rigorosamente uma metodologia de auditoria e refatoração baseada em 7 Pilares, aplicados a cada Módulo Lógico do sistema:

#	Pilar	Descrição
1	Clean Code	DRY, tipagem estrita, modularidade, SRP
2	Segurança / AppSec	Validação, sanitização, rate limiting, logging
3	Resiliência & Error Handling	Retry, circuit breaker, fallbacks, graceful shutdown
4	UX & Usabilidade	Loading states, acessibilidade, feedback visual, responsividade
5	Velocidade & Performance	Memoização, lazy loading, índices, virtualização
6	SEO & Semântica	Tags semânticas, meta tags, schema markup, breadcrumbs
7	Infraestrutura & Automação	Scripts PowerShell, seed, backup, deploy, health check
