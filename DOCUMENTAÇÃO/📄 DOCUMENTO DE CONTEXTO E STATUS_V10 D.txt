## 🆕 10. GERENCIAMENTO DE RESPOSTAS DO PREPOSTO — NOVA FUNCIONALIDADE

### 10.1 Descrição
O preposto agora pode **gerenciar as respostas** dos usuários que ele inseriu no sistema, podendo visualizar cada resposta e, se necessário, **solicitar revisões** com justificativa e anexos.

### 10.2 Estrutura de Dados

#### Modelo ReviewRequest (Backend)
```typescript
interface IReviewRequest {
  _id: string;
  companyId: ObjectId;        // Identifica a empresa (consulta rápida)
  responseId: ObjectId;        // Resposta original do usuário
  userId: ObjectId;            // Usuário que respondeu

O ResponsiveContainer do Recharts não funciona no contexto de impressão porque:

O ResizeObserver não é acionado durante a impressão

O container não tem altura definida → colapsa para 0px

O SVG interno não é renderizado
